## 📖 介绍

`nonebot-plugin-setu-now2` 是一个用于 NoneBot2 的插件，主要功能包括图片搜索、色图发送及相关消息处理。它支持异步请求，适配 OneBot V11 协议。

## 💿 安装

<details open>
<summary>使用 nb-cli 安装 (推荐)</summary>
